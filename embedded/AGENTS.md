# AGENTS.md

- 所有回复用中文。
- 本目录是久坐提醒器嵌入式项目；当前主线是 `firmware-idf/`，硬件默认按 `ESP32-S3 N16R8 CAM + OV5640CSP + SPI SSD1306 OLED + 蜂鸣器 + 按键` 处理。
- 后续只维护 `firmware-idf/`，不要恢复已删除的旧固件目录。
- 每次重要修改后必须同步更新 `README.md`，记录日期、修改内容、验证结果和下一步；`README.md` 同时作为开发进度和交接文档。
- 每次重大修改后必须检查 Git 忽略规则；如果新增了构建产物、缓存、临时目录或工具生成文件，要同步更新 `.gitignore`。
- 重大固件改动后优先运行 `idf.py build`；如果环境缺失，先补齐工具链，或在回复里明确未验证原因。
- 不要凭空确定摄像头引脚、板卡型号或硬件连接；未现场确认的内容必须标注为待核对。
- 不要把 `build/`、`build-*`、`managed_components/`、编辑器配置或其他缓存纳入交付产物。
- 文档保持可交接、简洁明确，不写无关说明。
