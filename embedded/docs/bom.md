# 元器件清单

| 类别 | 推荐型号 | 数量 | 说明 |

## 主控选择建议

当前主线选择 ESP32-S3 N16R8 CAM，因为用户已有该开发板，且 `N16R8` 提供 16MB Flash 和 8MB PSRAM，CAM 板能减少摄像头接线和供电调试成本。

备选顺序：

1. `ESP32-S3 N16R8 CAM + OV5640CSP`：当前主线，继续使用 PlatformIO + Arduino-ESP32。
2. `ESP32-S3-WROOM-1-MCN16R8 + 摄像头底板`：备选，需要额外处理摄像头硬件连接。
3. `ESP32-S3-EYE`：备选开发板，资料完整，适合视觉原型。
